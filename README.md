# agents

This project is set up for organizing agent exports and related notes.

## Structure

- definitions/ - contains one folder per agent
- each agent folder can include:
  - <agent-name>.json - the exported agent/workflow JSON used in n8n
  - README.md - notes or explanation for the agent

## Example

For example, your current agent is stored here:

- definitions/stock_news_morning_report/stock_news_morning_report.json
- definitions/stock_news_morning_report/README.md

This layout keeps the imported JSON and any companion documentation together.
